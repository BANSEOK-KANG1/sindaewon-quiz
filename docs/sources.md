# 기출 자료 수집 가이드

## 우선순위

1. **선배·동기·교회 신대원 준비 모임** — 연도별 PDF/필기 (가장 신뢰도 높음)
2. **다음 카페 에벤에셀** 등 — "총신 신대원 기출" 검색
3. **[Quizlet 총신 기출 세트](https://quizlet.com/kr/103234634/)** — 커뮤니티 플래시카드 (교차검증 필수)
4. **총신 cstsent** 공지·자료실 — 연도별 첨부 여부 수시 확인
5. **장신 출판부 성경문제집** — 장신 신대원 핵심 자료

## 앱에 넣기

1. `scripts/quizlet-bible.txt`, `scripts/quizlet-english.txt` 형식으로 정리
2. `node scripts/import-questions.js` 실행
3. `public/data/questions.json`이 갱신되면 앱 새로고침

저작권이 있는 출판물 전체를 자동 수집하지 말고, 합법적으로 확보한 내용만 입력하세요.
